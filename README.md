# e-commerce-app
# https://worried-tan-necklace.cyclic.app
# Product Functionality:

# Developed a MERN (MongoDB, Express, React, Node.js) stack e-commerce web application.
# Implemented user authentication with password hashing using bcrypt for enhanced security.
# Integrated the Braintree payment gateway for processing payments.
# Handled Cross-Origin Resource Sharing (CORS) for secure communication with the server.
# Utilized JSON Web Tokens (JWT) for user authentication and authorization.
# Managed form data using the Express-formidable middleware.
# Created SEO-friendly URLs by using the Slugify library.
# Modeled data and interacted with the MongoDB database using Mongoose.
# Designed the user interface using Bootstrap 5 for responsive and attractive layouts.
# Utilized Axios for making asynchronous HTTP requests to the server.
# Enhanced SEO by incorporating React Helmet to manage document head data.
# Implemented client-side routing with React Router DOM for a smooth user experience.
# Used React Hot Toast for user-friendly and customizable toasts.
# Enhanced the user interface with icons from React Icons and loading spinners from React Spinners.
# Tech Stack:

# Frontend:

# React (JavaScript library for building user interfaces)
# Bootstrap 5 (CSS framework for responsive design)
# Axios (JavaScript library for making HTTP requests)
# React Helmet (for managing document head data)
# React Router DOM (for client-side routing)
# React Hot Toast (for displaying toasts)
# React Icons (for adding icons)
# React Spinners (for loading spinners)
# Backend:

# Node.js (JavaScript runtime for server-side development)
# Express.js (Node.js web application framework)
# MongoDB (NoSQL database for data storage)
# Mongoose (ODM library for MongoDB)
# Bcrypt (for password hashing and security)
# Braintree (for processing payments)
# CORS (for handling Cross-Origin Resource Sharing)
# JSON Web Tokens (JWT) (for user authentication)
# Express-formidable (for managing form data)
# Slugify (for creating SEO-friendly URLs)
